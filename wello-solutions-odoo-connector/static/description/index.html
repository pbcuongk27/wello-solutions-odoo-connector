Export Partners type Company
Synchronize Odoo Partner to Wello company
  • Automatically create work order location for each customer
  • Create location based on address (Invoice, delivery, private, and other address)
Export Partners.Contact
Synchronize Odoo Contact of Partner to Wello contact
Export products
Synchronize Odoo product to Wello article, and product.category to Wello Article Category

1. Filter by Product category: When it is checked, step 4 'Select article categories to export' will be displayed, then the user can select the category that they want to import from Odoo to Wello

2. Export Product Prices: There are 3 options
  • None: No Odoo products prices will be sent to Wello

  • Default price: The Odoo product price lists will be sent to Wello. Please make sure that each price list contains a price for each product!

  • Pricelist: Based on Odoo rule, calculate the prices list of Odoo products on Wello
Export Work Order
Send completed work orders to Odoo
  • Sales Invoice: Send completed work orders to Odoo Sales Invoice

  • Sales Orders: Send completed work orders to Odoo Sales Orders

  • Let Odoo compute the products prices: If it is checked, we let Odoo compute the price on the fly (and all the rules of Odoo will be applied) when sending completed work orders to Odoo. If it is not checked, we use the price that has been computed in Wello



Installation




  • Odoo API URL: Be sure this url is reachable from outside your intranet. On the majority of the case, it looks likes https://YourOdooServer.odoo.com

  • Company DB: this is the dbname you see when login on Odoo. After sign in Odoo and clicking on the menu 'My Databases', you can find database that you want to connect on the screen 'Your Databases'

  • Username: the username of a valid-user

  • Password: the password of a valid-user



FAQs

Why connect your Odoo software with Wello?
Because this will strongly reduce the risk to have duplicate or onconsistent data. Wello is using the Odoo Service Layer to ensure the business process are followed
What are the requirements to connect Odoo to Wello?
  • You need to ensure the Odoo Api is running and reachable from your connector
  • You need to provide the Odoo DatabaseName, and valid user credentials
  • The user must be able to have full access to the entities we are working (ResPartner, Product, Orders, PriceList) in read/write
What are the configurations you need to setup in Odoo?
We don't need additional configuration in Odoo
What are the configuration you need to setup in Wello?
  • In Wello, make sure the billable hour types have the same code as their corresponding product in Odoo. So for each hour type, in the field 'code', fill in the corresponding article reference number

  • Dummy products are supported by this connector. When using dummy articles, the price and the description will be given when creating the Order

  • If you synchronize the products from Odoo to Wello, the 'sample' articles that are currently in the database and don't come from Odoo will be removed
